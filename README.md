# Agentic AI Chatbot using LangGraph, FastAPI, Streamlit, Groq, and Tavily Search

## Overview

This project is an Agentic AI Chatbot built using LangGraph, FastAPI, Streamlit, Groq LLMs, and Tavily Search. The application enables users to interact with intelligent AI agents capable of answering questions, performing web searches, and generating context-aware responses through a simple and interactive user interface.

The system follows a modular architecture consisting of a Streamlit frontend, a FastAPI backend, and a LangGraph-powered AI agent layer. Users can dynamically select AI models, customize system prompts, enable or disable web search capabilities, and receive real-time responses generated by large language models.

## Key Features

* Interactive chatbot interface built with Streamlit.
* FastAPI backend for handling API requests and agent orchestration.
* Agent-based architecture powered by LangGraph.
* Support for multiple LLM providers including Groq and OpenAI.
* Dynamic model selection through the user interface.
* Tavily Search integration for real-time web information retrieval.
* Customizable system prompts for defining agent behavior.
* Environment variable management using `.env` files for secure API key handling.
* Modular and scalable code structure for future enhancements.
* GitHub-ready project organization and deployment workflow.

## Project Workflow

1. User enters a query through the Streamlit interface.
2. The request is sent to the FastAPI backend.
3. The backend initializes the appropriate AI model and tools.
4. LangGraph orchestrates the agent workflow and tool execution.
5. Tavily Search is invoked when web search is enabled.
6. The LLM generates a contextual response using retrieved information.
7. The final response is returned to the frontend and displayed to the user.
